Configura GitHub

Per publicar automaticament, necessites un token personal de GitHub. Es guarda nomes al teu navegador, mai s'envia enlloc mes.

Crea'n un a github.com/settings/tokens amb scope repo.
Com obtenir el token?
1. Obre aquest enllac
2. A "Expiration", tria "No expiration" o la data que vulguis
3. Verifica que repo esta marcat
4. Clic a "Generate token"
5. Copia el token i enganxa'l aqui
